The What

A multi-page destination hub for the outdoor town of Ellicottville, New York, built in the mold of destination sites like Visit Park City, Discover Whistler, Aspen Snowmass, Discover Vail, and Eagle Outside — scaled for a single mountain village. It gathers everything a visitor needs into one branded home: two ski resorts (Holiday Valley and HoliMont), 140+ miles of mountain-bike trail, hiking, fishing, paddling, and nearby Allegany State Park — plus live weather, an events calendar, and full shop and dining directories.

The Why

Ellicottville has the resorts, the trails, and the visitors, but no single outdoor home online — the story is scattered across the two resorts, the chamber, and social feeds. This site pulls it into one branded destination, giving visitors a place to plan a day and local businesses a place to be found. It's a turnkey local media asset: an owned domain and social presence built into a finished, live site a local operator can take over and run from day one.

Ready to run

Finished and live

Already built, indexed, and accruing traffic — no build phase to fund and no long runway. An owner steps into a working property, not a project, and can start marketing it immediately.

The model

Free listings → traffic → paid features

Every shop and restaurant is listed free to build utility and traffic. Featured placements are held back as sellable inventory — the revenue line an owner steps into once the audience is proven.

04

Under The Hood

Static, fast, and cheap to run — with search and AI visibility engineered in.

SEO Full technical foundation. Unique title + meta description and a canonical tag on every page, Open Graph and Twitter cards with share images, a submitted sitemap.xml (11 URLs) and robots.txt, and clean semantic multi-page HTML. Rich structured data throughout: WebSite, Organization, 46× LocalBusiness, ItemList directories, 18× Event, Article, and Place schema. www redirects 308 → apex so link equity stays on one canonical host.
GEO
AI answer engines		 FAQ-driven, entity-rich. The home page and every guide carry FAQ blocks marked up with FAQPage / Question / Answer schema — plain-language, quotable answers that name the real places (Holiday Valley, HoliMont, Art Roscoe, Allegany). This is the primary lever for getting cited by AI search and answer engines, not just ranked by classic search.
Infrastructure Static site on Vercel. Plain HTML / CSS / JS with a shared stylesheet and script — no database, no server, near-zero maintenance. Hosted on Vercel (Pro team) with automatic HTTPS/SSL. Custom domain ellicottvilleoutside.com is live; DNS stays at DreamHost (apex A record + www CNAME to Vercel), email/MX untouched. Source is version-controlled in a private GitHub repo with off-machine backups. Only external calls are the NWS weather API and the Formspree form.
Analytics Two measurement layers, live. Vercel Web Analytics runs on all pages and is already accruing traffic history — the proof-of-audience a buyer needs. Google Search Console is verified (DNS TXT) with the sitemap submitted, giving indexing status and search-query data. GA4 can be dropped in with a single tag ID if a buyer wants it.
Running cost Minimal, and set up to transfer. The site is static — no database or backend to maintain, so there's nothing to keep running and hosting costs stay low. Domain, DNS, hosting, source repository, and analytics are all in place and transfer cleanly to a new owner. Ongoing cost is essentially the annual domain renewal.
